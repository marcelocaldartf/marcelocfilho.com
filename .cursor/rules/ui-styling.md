# UI & Styling

- **Tailwind CSS 4:** Use modern Tailwind 4 syntax.
- **Nuxt UI:** Use `U` components from Nuxt UI v3+.
- **Theme Variables:** Use CSS variables for colors (e.g., `--primary`).
- **Responsive:** Mobile-first approach using Tailwind breakpoints.
- **Icons:** Use `lucide-vue-next` or Nuxt UI's icon property.
